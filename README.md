# AI Content Marketing using UGC Ads – CafeAura

## 📌 About the Project

This project demonstrates how AI can be used to create high-converting User Generated Content (UGC) style advertisements for a real business.

Instead of traditional ads, this system generates content that feels natural, relatable, and engaging — similar to Instagram Reels and Shorts.

## ☕ Business Chosen

CafeAura – A cafe located in Visakhapatnam

## 🎯 Target Audience

- Students
- Coffee lovers
- Young professionals

## 🚀 Objective

- Generate UGC-style ad scripts using AI
- Create multiple hooks for marketing
- Follow Problem → Solution → CTA framework
- Produce content for Instagram Reels, Ads, and Shorts
- Build reusable prompt system

## 📱 Platform Adaptation

### Instagram Reels

- Fast-paced content
- Text overlays
- Trending audio
- Casual tone

### YouTube Shorts

- Slight storytelling
- Voice narration
- Engaging ending

### Ads (Paid Content)

- Strong hook
- Clear benefits
- Direct Call-To-Action (CTA)

## 🧠 Tools Used

- ChatGPT
- Claude
- Google Gemini
- CapCut (for video editing)

## 📊 Outcome

- Created high-converting UGC ad scripts
- Generated multiple engaging hooks
- Built reusable AI prompt system
- Produced ad-ready marketing content

## 💼 Use Case

This system can be used by:

- Local businesses
- Content creators
- Marketing agencies
- D2C brands
